    2 内容管理

    包括文章、专辑、曲目三个功能模块，对菩提心随身听APP显示的文章、专辑等进行设置。

    2.1 文章管理

    管理文章，对文章进行列表查看、新增、编辑、发布、排序、置顶、删除、搜索，具体如下：

    2.1.1 文章列表页面
    列表功能按钮介绍：
    发布/撤销发布：控制文章是否在APP显示，发状态的文章在APP显示，撤销发布后APP不显示；
    置顶/不置顶：置顶状态的文章显示在APP轮播面板里面；
    上移/下移：排序设置文章在APP的显示顺序。
    2.1.2 文章编辑页面
    上传文章封面介绍：
    点击上传封面按钮，弹出如下上传界面，在上传界面中点击选择文件选择上传文件或将上传的图片拖至上传区域，文件开始上传。
    等文件上传区域颜色转为淡绿色时，文件上传成功，点击“确定”按钮，返回文章编辑页面，如果上传正常即可在编辑页面看到上传的图片。
    2.1.3 文章查看页面

    查看页面预览文章在APP显示效果。同时可以用手机扫描二维码，预览页面在手机上的显示效果。

    2.2 专辑管理

    管理专辑，对专辑进行列表查看、新增、编辑、发布、排序、置顶、删除、搜索，具体如下：

    2.2.1 专辑列表页面
    列表功能按钮介绍：
    专辑列表功能按钮与文章列表功能按钮一致；
    专辑排序功能是在栏目内排序，跨栏目排序对APP显示没有影响，因此专辑列表的排序功能在搜索使选中栏目才会显示。
    2.2.2 专辑编辑页面
    添加音频介绍：
    点击编辑页面的“添加音频”按钮，弹出如下曲目列表，选中需要添加的曲目或文件夹，点击“确定”，即可将曲目加入对应的专辑；
    确定专辑音频后，点击“保存更改”，APP就可以播放对应专辑。
    2.2.3 专辑查看页面

    查看页面功能与文章管理查看页面功能一致。

    2.3 曲目管理

    管理曲目，包括曲目列表查看、曲目搜索、音频上传、曲目信息编辑、设置缩略图、曲目所属文件夹管理等功能。具体如下：

    2.3.1 曲目列表页面

    曲目列表页面功能包括：新建文件夹、编辑文件夹、删除文件夹，上传曲目、曲目列表搜索、编辑曲目信息、设置曲目缩略图、查看曲目、删除曲目。具体如下图所示：

    列表功能介绍：
    列表文件夹导航功能：进入子文件夹后，可以点击列头导航，跳转至任意一级父文件夹中；
    删除文件夹：删除文件夹时，会删除文件夹所有子文件夹及文件；
    曲目及文件夹命名：文件夹或曲目重名时，系统自动在名称后面补序号；
    列表搜索功能：搜索时只针对曲目有效，同时对搜索出的曲目可以跳转到曲目所属文件夹。
    2.3.2 编辑/新建文件夹

    点击列表中“新建文件夹”、“编辑文件夹名称”按钮，可弹出如下编辑文件夹界面，编辑好文件夹名称后点击“确定”按钮，即可新建/编辑文件夹。

    2.3.3 上传曲目

    点击列表中的“上传曲目”按钮，弹出入下图所示的曲目上传面板。

    点击“选择文件”按钮，选择上传的曲目后曲目开始上传，上传中的曲目显示上传进度条。等所有曲目上传完成后，点击“确定”按钮，完成上传并返回曲目列表页面。

    2.3.4 编辑曲目信息

    点击列表中的“编辑曲目信息”按钮，弹出如下图的曲目信息编辑面板，编辑好相关信息后点击“确定”按钮，即可保存。

    2.3.5 设置曲目缩略图

    设置缩略图同文章上传封面。

    2.3.6 曲目信息查看

    曲目信息查看页面可以查看曲目基本信息，试听曲目以及查看曲目缩略图。
